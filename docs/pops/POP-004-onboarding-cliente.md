# POP-004: Onboarding de Novo Cliente

## Objetivo
Garantir uma integração suave e profissional do novo cliente na Totum, estabelecendo expectativas claras e coletando todos os acessos necessários.

## Responsável
Project Manager (Liz) + Time Técnico (Jarvis)

## Gatilho
Contrato assinado + pagamento do setup confirmado.

## Procedimento

### FASE 1: Preparação (Dia 0 - Contrato assinado)
**Responsável:** Liz (30 min)

- [ ] Criar pasta no Drive: `Clientes/[Nome Empresa]`
- [ ] Criar grupo no WhatsApp: "[Empresa] + Totum"
- [ ] Adicionar: cliente, Israel, Liz, Jarvis, Pablo (se envolvido)
- [ ] Agendar kickoff meeting (próximos 3 dias úteis)
- [ ] Enviar email de boas-vindas com agenda

**Template email:**
```
Assunto: Bem-vindo à Totum! 🚀

Oi [nome],

Contrato assinado, pagamento confirmado. Agora é pra valer!

Próximos passos:
1. Kickoff meeting: [data/hora] - [link meet]
2. Checklist de acessos (em anexo)
3. Nosso grupo no WhatsApp: [link]

Na reunião vamos:
- Apresentar o time
- Definir metas do primeiro trimestre
- Alinhar cronograma
- Tirar dúvidas

Qualquer coisa, estamos no WhatsApp.

Abraço,
Liz
Totum
```

### FASE 2: Coleta de Acessos (Dias 1-3)
**Responsável:** Liz + Cliente

**Lista de acessos necessários:**

**Meta Ads:**
- [ ] Business Manager (convidar totum@grupototum.com como admin)
- [ ] Páginas do Facebook vinculadas
- [ ] Conta de anúncios com pagamento ativo

**Google Ads:**
- [ ] Conta Google Ads (ID da conta)
- [ ] Google Analytics (acesso proprietário)
- [ ] Google Tag Manager

**CRM/Automação:**
- [ ] Acesso ao CRM atual (se houver)
- [ ] Plataforma de email marketing
- [ ] WhatsApp API/Business

**Site/Landing Pages:**
- [ ] Acesso ao CMS (WordPress, etc.)
- [ ] Acesso ao domínio/registrar
- [ ] Acesso ao servidor/hosting (se aplicável)

**Outros:**
- [ ] Página do Google Meu Negócio
- [ ] LinkedIn da empresa
- [ ] Qualquer outro canal ativo

**Segurança:**
- Todos os acessos vão para o cofre do LastPass da Totum
- NUNCA salvar senhas em planilhas ou documentos
- Ativar 2FA sempre que possível

### FASE 3: Kickoff Meeting (Dia 3-5)
**Duração:** 60 min
**Participantes:** Cliente + Time Totum completo

**Agenda:**
1. **Apresentações** (5 min) - Quem é quem
2. **Revisão do contrato** (5 min) - O que foi contratado
3. **Entendimento do negócio** (15 min) - Cliente explica operação atual
4. **Metas e KPIs** (15 min) - O que sucesso parece em 90 dias
5. **Cronograma** (10 min) - Primeiras entregas
6. **Dúvidas** (10 min) - Tirar qualquer pendência

**Entregáveis pós-kickoff:**
- [ ] Ata da reunião enviada em 24h
- [ ] Planilha de metas compartilhada
- [ ] Cronograma no Asana/Monday

### FASE 4: Setup Técnico (Dias 5-15)
**Responsável:** Jarvis + Pablo

**Semana 1:**
- [ ] Auditoria técnica inicial (SEO, Analytics, Ads)
- [ ] Configuração de pixels e tags
- [ ] Setup/otimização de campanhas iniciais
- [ ] Integração CRM (se aplicável)

**Semana 2:**
- [ ] Primeiras campanhas no ar (teste)
- [ ] Dashboard de relatórios configurado
- [ ] Fluxo de notificação de leads pronto
- [ ] Validação com cliente

### FASE 5: Primeira Revisão (Dia 30)
**Responsável:** Israel + Liz

- [ ] Reunião de 30 min: como foi o primeiro mês
- [ ] Relatório de performance vs metas
- [ ] Ajustes de estratégia se necessário
- [ ] Coleta de feedback do cliente

## Documentação Obrigatória

Cada cliente deve ter na pasta do Drive:
- [ ] Contrato assinado
- [ ] Briefing completo
- [ ] Acessos (no LastPass, não no Drive)
- [ ] Ata do kickoff
- [ ] Planilha de metas
- [ ] Cronograma de entregas

## Checklist de Sucesso do Onboarding

Após 30 dias, verificar:
- [ ] Todas as campanhas contratadas estão no ar
- [ ] Cliente recebe relatórios semanais
- [ ] Lead flow está funcionando (testar)
- [ ] Cliente responde em < 24h no grupo
- [ ] Não há pendências de acesso
- [ ] Primeira reunião de revisão realizada

## KPIs de Onboarding
- Tempo até primeiro lead: < 14 dias
- Cliente "ativação": participa do grupo, responde rápido
- NPS do onboarding: > 8

## Riscos Comuns e Mitigação

| Risco | Mitigação |
|-------|-----------|
| Cliente não passa acessos | Cobrar diariamente, lembrar que delay atrasa resultados |
| Cliente some após assinar | WhatsApp + ligação, entender objeção |
| Expectativa irreais | Revisar no kickoff, documentar metas realistas |
| Demanda fora do escopo | "Ótima ideia, vamos incluir na próxima fase" |
